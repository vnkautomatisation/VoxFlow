# /public/logo — Assets logo VoxFlow

## Fichiers

| Fichier            | Usage                              | Dimensions  |
|--------------------|------------------------------------|-------------|
| voxflow-logo.svg   | Source master — toujours préférer  | vectoriel   |
| voxflow-icon.svg   | Icône carrée (apps, badges)        | vectoriel   |
| logo-sm.png        | Petits espaces (email, docs)       | 240×60      |
| logo-md.png        | Usage standard                     | 480×120     |
| logo-lg.png        | Haute résolution, impression       | 960×240     |

## Règle
**Ne jamais recréer le logo manuellement.**
Voir `LOGO_STANDARD.md` à la racine du projet.

## Régénération
Si le logo doit évoluer, modifier le SVG master `voxflow-logo.svg`
puis relancer le script de génération PNG.
